> **Multi-agent orchestration for [kimi-cli](https://www.kimi.com/code). Zero learning curve.**

**Don't learn kimi-cli. Just use OMK.**

OMK transforms kimi-cli into a team of 36 specialized agents working together to complete your tasks with unprecedented speed and quality.

[Installation](#installation) • [Quick Start](#quick-start) • [Documentation](docs/) • [Examples](examples/)

## ✨ Features

- **🎯 Zero Configuration** - Works out of the box
- **🤖 36 Specialized Agents** - From architects to security reviewers
- **⚡ Automatic Parallelization** - Up to 6 agents working simultaneously  
- **🔄 Persistent Execution** - Won't stop until the job is done
- **💰 Cost Optimization** - Smart routing saves 30-50% on tokens
- **🗣️ Natural Language** - Just describe what you want
- **🤝 Multi-AI Support** - Integrates Claude, Codex, and Gemini

## 🚀 Quick Start

### Prerequisites

- [kimi-cli](https://www.kimi.com/code) installed
- Node.js >= 20.0.0
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/sergionsantos/oh-my-kimi.git
cd oh-my-kimi

# Run the setup wizard
/omk-setup
```

### Your First OMK Session

```bash
# Build something autonomously
$autopilot "build a REST API for managing tasks"

# Fix errors in parallel
$team 3 "fix all TypeScript errors"

# Deep research
$sciomc "analyze authentication patterns"
```

**That's it!** Everything else is automatic.

## 🎭 Magic Keywords

Just type naturally. OMK understands:

| Keyword | What It Does | Example |
|---------|--------------|---------|
| `autopilot` | Full autonomous execution | `autopilot: build a todo app` |
| `team` | Multi-agent coordination | `team 3: fix all errors` |
| `ralph` | Won't stop until complete | `ralph: refactor auth` |
| `plan` | Strategic planning | `plan this feature` |
| `sciomc` | Scientific research | `sciomc: analyze patterns` |
| `ask` | Ask other AIs | `ask claude "review this"` |
| `ccg` | Tri-model synthesis | `ccg: comprehensive review` |
| `cancel` | Stop execution | `cancel` |
