## [1.0.0] - 2026-03-18

### Added
- Initial release of Oh My Kimi
- 36 specialized agents covering all development needs
- 32 workflow skills for complete project lifecycle
- Multi-agent orchestration with parallel execution
- Natural language interface with magic keywords
- State persistence and session management
- External AI integration (Claude, Codex, Gemini)
- Comprehensive documentation and examples
